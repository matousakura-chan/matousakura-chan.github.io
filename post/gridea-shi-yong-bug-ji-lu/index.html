Gridea 使用 BUG 记录

2021-03-06 1 min read

Gridea 编辑器无法正常使用

使用内置的编辑器出现前面段落无法删除的情况，解决方案为用 Gridea 生成文章后在源文件目录用 VSCode 或 Typora 打开。

Typora 编辑后出现所有文件消失

原因应该是原作者代码的问题，出现文章的 md 文件缺少开头：

title: 'XXX'
date: 2021-03-06 13:52:10
tags: []
published: false
hideInList: false
feature: 
isTop: false

将直接出现异常导致无法读取真个根目录的文件。

建议还是使用 VSCode 编辑，能查看源代码，Typora容易不兼容。

    • Gridea 编辑器无法正常使用
    • Typora 编辑后出现所有文件消失
